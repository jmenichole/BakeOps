# Security Policy

## Supported Versions

We currently support and provide security updates for the following versions of BakeBot:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of BakeBot seriously. If you believe you have found a security vulnerability, please report it to us as soon as possible.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email your report to: **jmenichole007@outlook.com**

In your report, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact if exploited.

## Disclosure Policy

When a vulnerability is reported, we will:
1. Acknowledge receipt of your report within 48 hours.
2. Provide an estimated timeline for a fix.
3. Notify you once the vulnerability has been resolved.

We ask that you give us a reasonable amount of time to resolve the issue before making any information public.

Thank you for helping keep BakeBot secure for all our bakers!
