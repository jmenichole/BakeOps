EXECUTIVE SUMMARY

BOT — Baked On Time

BOT is an AI-powered operations and sales platform for custom cake businesses. We turn informal customer ideas into production-ready designs, automated quotes, ingredient plans, and a live fulfillment dashboard — compressing hours of manual back-and-forth into seconds.

Core Atomic Units: Idea → AI Design → Quote → Booking → Production Timeline → Ingredient Batch Plan → Delivery Status.

Initial MVP Live: Public waitlist capture + Supabase backend. Next milestone: closed beta with 15 bakers to validate design fidelity + quoting accuracy.

PROBLEM

The Manual Bottleneck

  • Custom orders start as scattered DMs, emails, screenshots.
  • Quoting requires manual ingredient estimation + tier sizing.
  • Scheduling multi-day prep is tracked in notebooks or spreadsheets.
  • Design inspiration rarely matches operational constraints (size, supports, timing).
  • Lost revenue: slow response time → customer churn to generic bakeries.
SOLUTION

Instant From Inquiry to Production

AI Design Engine

Generates high-quality cake mockups in the baker’s own style (trained on portfolio photos).

Smart Quoting

Dynamic pricing using size, tiers, support requirements, premium ingredient multipliers.

Baker Dashboard

Live prep timeline, batchable ingredient list, status bars for multi-day tasks (bakes, fillings, decor).

Customer Experience

Fast branded PDF or link: design + quote + payment button within minutes.

MARKET

Target & Expansion

  • Beachhead: Independent custom cake studios (US & Canada) ~35–40K businesses.
  • Serviceable Initial: 8–10K digitally active bakers with Instagram storefront presence.
  • Expansion: Wedding specialty bakeries → pastry artists → boutique dessert studios.
  • Future Horizontal: Floral arrangements, event charcuterie boards, custom confectionery.
BUSINESS MODEL

Revenue Streams

  • SaaS Subscription: $59 / $129 tiers (solo vs studio) — design volume + batch tools.
  • Transaction Add-On: 1.5% success fee on processed bookings (Stripe).
  • Design Credits: Additional AI style retraining / seasonal template packs.
  • Marketplace (Phase 3): Upsell premium supports, stands, decor materials.

Year 2 Target: 1,500 paying baker accounts @ blended ARPU $72 → ~$1.3M ARR + transaction layer.

TECH STACK

Architecture Snapshot

  • Frontend: Static HTML/CSS/JS (GitHub Pages) → ultra low overhead.
  • Backend: Supabase (Postgres + Auth + Edge Functions) for waitlist + analytics.
  • AI Layer (Upcoming): Fine-tuned image generation using baker portfolio embeddings.
  • Scheduling Engine: Task graph for multi-day dependencies (bake → cool → fill → stack → decorate).
  • Scalability: Stateless Edge Functions for rapid vertical feature iteration.
